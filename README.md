Giới Thiệu

Dự án này là hệ thống quản lý dịch vụ xe buýs, được xây dựng bằng Spring Boot, MySQL và Spring Security. Hệ thống cho phép quản lý tuyến xe, lịch trình và vé xe.

Công Nghệ Sử Dụng

Backend: Spring Boot, Spring Data JPA, Spring Security

Database: MySQL

Frontend: (Chưa xác định - có thể dùng React/Angular)

Authentication: JWT
